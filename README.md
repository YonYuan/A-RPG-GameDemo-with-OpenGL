# 一个简单的OpenGLRPG游戏 

一个2D迷宫游戏。

实现包含底层碰撞判定，通过判定层颜色判断边界

瓷砖地图的实现，贴图绘制

角色移动，按键输入和反馈

迷宫的随机生成，迷宫生成算法

简单的剧情流程
